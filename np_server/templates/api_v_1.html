Введение

Ключевые идеи.

Проект по поиску и трекингу мобильных устройств, начал развиваться с идеи того что в настоящем существует проблема

потерянных устройств. Существующие решения не плохи, но не дают хорошего решения.

Мы поставили себе цель создания общего API, более универсального, для любых типов мобильных устройств: Телефоны, Планшеты, Ноутбуки, Флеш карты и т.д.

Акцент делается на слове "универсальный". Что мы понимаем под данным словом, это не просто решение для одного типа устройства

по большему счету это глобальная проблема нашего времени. Хотя это кажется довольно утопической мыслью, мы решили развивать данное направление.

Второе направление которое мы развиваем поиск мобильных устройств. Существующие решения не позволяют найти свой телефон в быстрые сроки.

Многие вещи зависят от обстоятельств благодаря которым можно найти устройства. Органы правоохранительной власти не всегда способны во время отреагировать

на вашу просьбу, или иногда это просто становится затруднительным по различным причинам или обстоятельствам.

Иногда вы просто можете оставить ваше устройство где-нибудь в ванной, а иногда в общественном транспорте. В любом случае очень хочется иметь инструмент

приближающий нас к тому чтобы разрешить проблему в ближайшее время. Это довольно раздражает, не так ли?

Наши возможности

GET для получения и поиска данных

POST для добавления данных

PUT для обновления данных

DELETE для удаления данных

Определение API

Наше API содержит следующие методы:

Метод | URL | Действие

GET /api_v_1/ Возвращает все ваши устройства

GET /api_v_1/device/2 Возвращает данные по устройству по его ключу (api key)

POST /api_v_1/device Добавляет устройство

PUT /api_v_1/device/2 Обновляет местоположение девайса по его ключу

DELETE /api/device/2 Удаляет местоположение девайса по его ключу
